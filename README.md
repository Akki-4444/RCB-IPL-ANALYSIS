# RCB-IPL-ANALYSIS
Project Overview
This project analyzes the performance of Royal Challengers Bangalore in the Indian Premier League (IPL) using historical match data.
The goal is to extract insights about team performance, player contributions, and match outcomes using data analysis techniques.

🎯 Objectives
Analyze RCB’s performance across seasons
Identify key players and their impact
Study win/loss patterns
Understand factors affecting match results

🛠️ Tools & Technologies Used
SQL (for data querying)
Excel / Power BI (for visualization)

📊 Key Analysis Performed
Total matches played, wins, and losses
Top run scorers and wicket takers
Toss impact on match results
Venue-wise performance
Season-wise trend analysis

📈 Key Insights
RCB performs better when batting second (example insight — update based on your data)
Certain players consistently contribute to wins
Toss decision has a noticeable impact on match outcomes
Performance varies significantly across venues
